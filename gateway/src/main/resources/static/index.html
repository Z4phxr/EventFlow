EventFlow - API Gateway (status)

Ta strona jest serwowana przez API Gateway i ułatwia szybkie sprawdzenie zdrowia oraz routingu.

Podstawowe linki

  • /actuator/health - ogólny health endpoint gatewaya
  • /actuator/gateway - lista tras skonfigurowanych w gateway
  • /api/events - przykładowe proxy do `event-service` (GET lista)
  • /api/notifications - proxy do `notification-service`
